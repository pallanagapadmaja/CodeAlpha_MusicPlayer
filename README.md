# CodeAlpha_MusicPlayer

## Overview
Music Player is an interactive web-based application developed as part of the CodeAlpha Frontend Development Internship. It allows users to play, pause, switch songs, and control music playback through a simple and responsive interface.

## Features
- Play music
- Pause music
- Next song
- Previous song
- Display song title
- Responsive design
- Interactive controls
- Smooth user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
CodeAlpha_MusicPlayer/
│── musicplayer.html  

## File Description

### musicplayer.html
This single file contains the complete Music Player application including HTML structure, CSS styling, and JavaScript functionality.

It includes:

- Music player heading/title
- Song display section
- Play button
- Pause button
- Next song button
- Previous song button
- Audio source integration
- Real-time playback control
- Responsive design for different screen sizes
- Interactive button functionality
- Smooth styling enhancements

## Functionalities

### Play Music
Starts the selected song.

### Pause Music
Pauses the currently playing song.

### Next Song
Moves to the next song.

### Previous Song
Returns to the previous song.

### Song Display
Shows the currently playing song name.

### Responsive Interface
Adjusts layout for different devices.

## How to Run the Project
1. Download the project files  
2. Keep audio files in the same folder  
3. Open musicplayer.html in browser  
4. Use controls to play music
